Stenogsaurus

Undergraduate Real World Project

(Python 3, SQL, Twitter API)

Written in Python 3; this group project was an encrypted and encoded messaging-app, implementing RSA encryption; stenographic methods; an SQLite database; and the Twitter API.

I designed and implemented the stenographic functions - using the Pillow library to encoode ciphertext binary strings into RGB pixel values, effectively 'hiding' it in the file. The decoding algorithm could retrive the embedded message from the image, returning it from the user.

As a university module, assessment was done via weekly presentations and an independent project report. Overall I achieved a First at 71%.

Visual Keyboard

A Level Computer Science Project

(C#, SQL, Emgu-CV)

My individual project was an ASL sign-language interpreter written in C#. Utilizing computer vision via Emgu-CV, my project was able to recognise basic ASL gestures via the webcam.

The solution refined images using the Watershed algorithm, and analysed the images using Haar cascades and the Viola-Jones algorithm. The user data was stored in a locally hosted MySQL server, associated with the user account.
